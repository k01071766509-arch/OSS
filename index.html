OSS 개발과 협업

Git, GitHub, Notion 핵심 정리 웹페이지

Git Branch

  • Main: 최종 배포 브랜치
  • Develop: 개발 내용 통합 브랜치
  • Feature: 새로운 기능 개발 브랜치
  • Release: 배포 전 테스트 브랜치
  • Hotfix: 배포 후 긴급 수정 브랜치

Git 명령어

  • git branch: 브랜치 목록 확인
  • git checkout -b 이름: 브랜치 생성 + 이동
  • git merge: 브랜치 병합
  • git status: 현재 상태 확인

GitHub

GitHub는 Git 기반의 원격 저장소 및 협업 플랫폼이다.

Notion

Notion은 문서 작성, 일정 관리, 데이터베이스, 협업을 지원하는 올인원 도구이다.
